Generate Accurate Academic References

This website was developed to help Independent Institute of Education (IIE) students create accurate references according to the IIE Harvard Anglia Referencing Style.

Many students lose marks because of incorrect referencing rather than weak academic content. This platform simplifies the referencing process through guided forms, reference examples, formatting tips and in-text citation guidance.

Why This Website Was Developed

Generic online citation generators often do not follow the exact requirements of the IIE Harvard Anglia Referencing Style.

This website focuses specifically on helping students generate accurate references and in-text citations that comply with institutional referencing guidelines.

Website Features

  • Generate academic references automatically.
  • Reference multiple source types.
  • Copy generated references directly into assignments.
  • Access reference examples and formatting guidance.
  • Learn correct in-text citation techniques.
  • Reduce referencing errors in academic work.

Supported Source Types
